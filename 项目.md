## Better-scroll

`npm install better-scroll --save`

[better-scroll](https://zhuanlan.zhihu.com/p/27407024)



* 上拉加载刷新
* 下拉加载更多

```javascript
const bs = new BScroll(document.querySelector(".wrapper"),{
    probeType:2,pullUpLoad:true
})
bs.on('scroll',position=>{
    // console.log(position);
})
bs.on('pullingUp',()=>{
    console.log('ssss');
    setTimeout(()=>{
        bs.finishPullUp()
    },2000)
})
```
